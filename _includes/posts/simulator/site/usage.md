# Usage

~~~
Usage: as-simulator [options] The list of simulation files to execute
  Options:
    -batch_size
       Transfer output batch size
    -data_store
       Directory path for data store
    -discovery
       Discovery URL
    -distribution_role
       Distribution role (none, leech, seeder)
    -?, -help
       Print this help message
    -listen
       Listen URL
    -member_name
       Member name
    -metaspace
       Metaspace name
    -no_exit
       Do not shut down after the Eclipse application has ended
    -operation
       Space operation (get, load, none, partial, put, take)
    -rx_buffer_size
       Receive buffer size
    -wait_for_ready_timeout
       Wait for ready timeout
    -worker_thread_count
       Worker thread count
    -writer_thread_count
       Number of writer threads
~~~
